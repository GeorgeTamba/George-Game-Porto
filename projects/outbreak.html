Outbreak

3D FPS Zombie Game

Project Overview

A first-person shooter where players must survive waves of relentless zombies. The objective is to navigate through danger zones, fend off the undead, and reach designated safe zones to stay alive.

Documentation

Key Features

  • Finite State Machine (FSM) for Zombie AI
  • Dynamic Pathfinding System
  • State-Specific Behavior
  • Combat System
  • Smooth Transition Handling

Technical Details

  • Role: Game Programmer & Level Designer
  • Team Size: 3 People
  • Time Frame: 1 Week
  • Engine / Tools: Unity
  • Game Platform: PC
  • Released: February, 2025

My Contributions / What I Have Learned

Designing the map of the game is my role in this project. Designing the path for player to find while trying to survive from the zombies' attack. I also contributed on FSM logic, zombies transition between states: Idle, Chase, Attack, and Dead so the animation must also played when the FSM transitionend
