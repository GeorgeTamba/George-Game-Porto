ZombieTop

2D Game

Project Overview

A game where a soldier must escape from a horde of zombies across the rooftops of several buildings. Armed with a gun, he must clear the path and keep moving toward a rescue helicopter waiting at the end of each level.

Gallery

Key Features

  • Fast-paced 2D platformer gameplay
  • Multiple levels with increasing difficulty
  • Various zombie types with unique behaviors
  • Weapon upgrades and power-ups
  • Score system and achievements

Technical Details

  • Engine: Unity
  • Platform: Windows
  • Development Time: 3 months
  • Team Size: Solo project
