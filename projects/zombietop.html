ZombieTop

2D Game

Project Overview

A game where a soldier must escape from a horde of zombies across the rooftops of several buildings. Armed with a gun, he must clear the path and keep moving toward a rescue helicopter waiting at the end of each level.

Documentation

Key Features

  • Fast-paced 2D platformer gameplay
  • Multiple levels with increasing difficulty
  • Various zombie types with unique behaviors
  • Power-ups such as health boost
  • Kill counter as the game scoring

Technical Details

  • Role: Game Programmer, Game Designer & Game Artist
  • Team Size: 1 Person
  • Time Frame: 1 Week
  • Engine / Tools: Unity & Aseprite
  • Game Platform: PC
  • Released: February, 2025

My Contributions / What I Have Learned

I made the game character, enemy character and buildings asset from scratch using Aseprite a sprite and pixel editor. I create an zombie NPC that detect player character using raycast for zombie to indentify if they see the player and if they are on the same building as the player. So mainly I contributed on all the aspects of the game, starting from gameplay mechanics, game designs and character design.

Doing this project, which is my first personal project it was very frustating due to the lack of the knowledge. But in the process of making this project I learned a lot of game mechanics and level design, it helps me to get the flow of developing a game.
